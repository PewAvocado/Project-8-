# webpack-starter
My main template for projects

This is going to be my main template so I can start any project faster.

How to use?
1. Start with : npm install
That will install the needed plugins and make sure everything can run proeprly.

2. Use : npm start
This will start the live server so you can see every change you start making.

3. Crear the new branch gh-ages, using git branch gh-pages

4. Now we will make sure eveything is added to github (while in branch main),using git add, git commit and git push

5. Finally, time to use the scripts , in this order that has been set already by myself.

npm run deploy1
npm run build
npm run deploy2
npm run deploy3
git checkout main

You can check what each script does in the package.json if wanted.

That would be it for adding changes to your repositories.
